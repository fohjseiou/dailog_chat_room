### v0.2.0 (2026-03-09) - 测试基础设施

### v0.1.0 (2026-03-09) - MVP 发布

**核心功能**
- ✅ 智能对话系统
  - 多轮对话支持
  - SSE 流式响应
  - 意图识别 (法律咨询/问候/一般对话)
  - RAG 检索增强生成

- ✅ 知识库管理
  - 文档上传 (PDF/DOCX/TXT)
  - 智能分块和向量化
  - 分类管理 (法律法规/案例分析/合同范本/司法解释)
  - 批量操作

- ✅ 会话管理
  - 会话历史记录
  - 自动摘要生成
  - 会话列表和切换

- ✅ 用户界面
  - 响应式设计
  - 移动端适配
  - 骨架屏加载状态
  - 思考状态指示器

**后端技术栈**
- FastAPI + Python 3.11
- DashScope Qwen LLM
- LangGraph Agent 编排
- ChromaDB 向量存储
- SQLite/PostgreSQL 数据库

**前端技术栈**
- React 18 + TypeScript + Vite
- Zustand 状态管理
- React Router v6
- CSS Variables 设计系统

## 技术债务

### 优先级：高

| 项目 | 影响 | 计划版本 |
|------|------|----------|
| API 集成测试 | 测试覆盖不足 | v0.3.0 |
| E2E 测试 | 缺少端到端测试 | v0.3.0 |
| 错误日志集中化 | 调试困难 | v0.3.0 |
| 性能监控 | 性能问题难定位 | v0.4.0 |
| 数据库迁移工具 | 部署风险 | v0.3.0 |

### 优先级：中

| 项目 | 影响 | 计划版本 |
|------|------|----------|
| 前端状态管理优化 | 复杂状态管理 | v0.4.0 |
| API 版本控制 | 兼容性问题 | v0.5.0 |
| 缓存策略优化 | 性能提升 | v0.4.0 |
| 前端性能优化 | 加载速度 | v0.3.0 |

### 优先级：低

| 项目 | 影响 | 计划版本 |
|------|------|----------|
| TypeScript 严格模式 | 类型安全 | v0.4.0 |
| ESLint 规则完善 | 代码质量 | v0.4.0 |
| 文档自动生成 | 维护效率 | v0.5.0 |
| Docker 镜像优化 | 部署效率 | v1.0.0 |
